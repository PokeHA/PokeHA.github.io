《Linux从入门到精通》笔记 第6章 文件目录管理

6.1 Linux文件系统的架构

  • Linux没有盘符，也就是C盘D盘。
  • Linux先有根目录，再有分区，可以将不同的分区挂载到不同的子目录上。
  • 对于没有挂载目录的分区，统一放在根分区中。

6.3 建立文件和目录

  1. 建立目录：mkdir
    • 例子：
      1. mkdir document picture
        • 在当前目录下创建 document和picture两个文件夹。
      2. mkdir ~/picture/temp
        • 在指定目录下创建指定文件夹。
        • 注意路径中的每一个父目录都必须存在。
      3. mkdir -p ~/tempx/job
        • 加上了 -p 选项，可以新建一个完整的路径。
  2. 建立一个空文件：touch
    • touch 文件名。
    • 可以覆盖文件。

6.4 移动、复制和删除

  1. 移动和重命名：mv
    • 例子：
      1. mv hello bin/
        • hello 文件移动到 bin/ 下。
        • 注意文件夹后面一定要加斜杠 /，不然就是文件。
      2. mv Photos/ 桌面/
        • Photos/ 文件夹移动到 桌面/ 下。
    • 文件重复问题：
      1. mv 默认会覆盖重复文件。
      2. mv -i 会在覆盖前提示，y 覆盖，n 跳过。
      3. mv -b 发现重复时，在重复的文件的名字后面自动加上 ~。
    • 重命名：
      1. mv fileA fileB 将 fileA 重命名为 fileB。
      2. 重命名实际上就是移动文件的同时重命名。
  2. 复制文件和目录：cp
    • 例子：
      1. cp test.php test/
        • 将 test.php 复制到 test/ 目录下。
        • 默认覆盖同名文件。
    • cp -i 发现重名文件时进行提示。
    • cp -r 普通的 cp 在复制的时候只复制文件，会跳过文件夹，cp -r 会复制所有文件和文件夹，以及文件夹里的所有子目录。
  3. 删除目录和文件：rmdir 和 rm
    • rmdir：
      • 用于删除文件夹，但只能删除空的文件夹，基本没用。
    • rm：
      • rm 文件名 只能删除文件。
      • 删除被保护文件时会提示。
      • rm -i 文件名 删除所有文件时都会提示。
      • rm -f 文件名 和 rm -i 相反，删除被保护文件时不提示。
      • rm -r 目录 递归删除所有文件和目录。
      • rm -rf 无提示地删除当前目录下的所有目录和文件，慎重使用。

6.5 文件和目录的权限

  • Linux是多用户操作系统，可以给不同用户设置权限。
  1. 权限设置针对的用户：
    • Linux中的三种用户：
      1. 文件所有者（属主）
        • 通常是文件创建者，也可以通过root转让给别人。
      2. 文件属组用户
        • 文件的属组是一组用户的集合。
      3. 其他人
        • 权限最低。
      4. 超级管理员root单独算。
  2. 需要设置哪些权限：
    • 三种权限：
      1. 读取 r
      2. 写入 w
      3. 执行 x
    • 文件的三种权限：
      1. 读取：打开查看文件的内容。
      2. 写入：修改文件内容。
      3. 执行：执行可执行文件。
    • 目录的三种权限：
      1. 读取：用户能否列出目录中的内容。
      2. 写入：在目录中创建、删除和重命名文件。
      3. 执行：用户能否进入目录。
  3. 查看文件和目录的属性：
    • 使用 ls -l /bin/login 可以查看文件的属性。
    • 使用 ls -ld /etc/ 显示 etc 文件夹的属性。-d 表示显示选择的当前文件夹。
  4. 改变文件所有权：chown 和 chgrp
    • chown [OPTION] … [OWNER][:[GROUP]] FILE…
      • 将 FILE 的属主改为 OWNER，属组改为 GROUP。
    • sudo chown -R lewis iso/
      • 将 iso/ 和其中的所有子目录全部给 lewis，-R 表示递归操作。
    • chgrp 可以单独更改文件的属组。
  5. 更改文件权限：chmod
    • chmod u+x days 给属主增加 days 的执行权限。
    • chmod a-x days 给所有角色删除 days 的执行权限，a 代表所有角色（即 ugo）。
    • chmod ug=rx, o=r days 给属组和属主读和执行权限，给其他人读权限。
    • chmod o=u days 将属主的权限赋予给其他人。

6.6 文件类型

  • Linux中的所有内容都表示为文件，共有七种文件类型。
    • 使用 ls -l 可以查看文件类型，第一列的第一个字符表示文件类型：
      • d 表示目录文件，- 表示普通文件。
  1. 建立链接（符号链接）：ln
    • ln -s TARGET LINK_NAME
      • 为文件 TARGET 创建软链接 LINK_NAME。
      • 加上 -s 选项创建软链接。

6.7 输入输出重定向和管道

  1. 输出重定向：
    • 程序的默认输出地方叫做 stdout（标准输出），标准输出指向显示器。
    • 输出重定向可以把程序的输出转移到另一个地方：
      • 例子：ls > ~/ls_out
        • 将 ls 指令的输出放到 ~/ls_out 文件中。
        • 如果文件已存在，会直接覆盖。用 >> 追加内容。
  2. 输入重定向：
    • 程序默认接收输入的地方叫 stdin（标准输入）。
      • 例子：cat < days 从 days 文件中接收输入并打印出来。
      • Here Document：cat << EOF 表示只从键盘获取输入，EOF 为用户指定的终止符。
  3. 管道：
    • 管道将一条命令的输出连接到另一条命令的输入，使用 | 关键字：
      • 例子：ls | grep ay
        • 先由 ls 列出当前目录下的全部文件名，然后将结果发给 grep，找出包含 “ay” 的文件。

6.8 小结

  1. Linux将文件系统挂载到目录下，根目录 / 是最初建立的文件系统。
  2. mkdir 创建新目录。
  3. touch 创建空文件。
  4. mv 移动并重命名文件和目录。
  5. cp 复制文件和目录。
  6. Linux文件角色分为属主、属组、其他人。
  7. Linux文件权限分为 rwx。
  8. ls -l 可以显示文件的所有属性，ls -ld 显示当前目录的属性。
  9. chown 更改文件的属主和属组。
  10. chmod 更改文件权限。
  11. Linux有七种文件类型。
  12. ln -s 建立软链接。
  13. 输入重定向。
  14. 输出重定向。
  15. 管道。
